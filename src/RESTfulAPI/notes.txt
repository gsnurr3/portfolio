Initial EF Core Migration:

    AppDbContext:
    dotnet ef migrations add InitialCreate_App --context AppDbContext --output-dir Persistence/Migrations/App -s src/RESTfulAPI -p src/RESTfulAPI
    dotnet ef database update --context AppDbContext -s src/RESTfulAPI -p src/RESTfulAPI

    LogDbContext:
    dotnet ef migrations add InitialCreate_Log --context LogDbContext --output-dir Persistence/Migrations/Log -s src/RESTfulAPI -p src/RESTfulAPI
    dotnet ef database update --context LogDbContext -s src/RESTfulAPI -p src/RESTfulAPI

Push DB Changes to Azure DB:
1. Add & Commit:

dotnet ef migrations add AddNewColumn 
git add Persistence/Migrations 
git commit -m "Initial Creation"

2. Produce idempotent SQL script:

dotnet ef migrations script --context AppDbContext --output ./deploy/01-AppDb-migrations.sql --idempotent

3. Review AppDb-migrations.sql under /deploy

4. Deploy

az login --use-device-code --tenant df5a51fd-34f1-4f7f-8dd6-967ccaac0efc
